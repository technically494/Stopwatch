#  Stopwatch Web App

##  Project Description

This is a simple stopwatch web application built using HTML, CSS, and JavaScript.
It allows users to measure time with start, pause, reset, and lap functionality.

##  Features

• Start and pause timer
• Reset stopwatch
• Lap time recording
• Real-time display (HH:MM:SS format)

##  Technologies Used

• HTML
• CSS
• JavaScript

##  How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
